# Release Gates

Parent: [[docs/RootCellar/00-Program/RootCellar Master Plan]]

## Gate A Core Viability Beta
- XLSX no-repair >= 80% on active corpus.
- Editing loop stable for core workflows.
- Macro alpha with capability prompts and audit logs.
- Observability critical trace completeness >= 90%.

## Gate B Replacement Credibility Beta
- XLSX no-repair >= 88%.
- Golden formula parity threshold met for scoped families.
- Compatibility panel fully wired to parser/calc statuses.
- CLI reproducibility checks integrated in nightly CI.

## Gate C Release Candidate
- XLSX no-repair >= 92%.
- Security validation complete with no critical findings.
- Performance SLOs sustained for 2 weeks.
- Incident and support playbooks validated.

## Required Artifacts For Gate Reviews
- Compatibility trend report.
- Performance benchmark package.
- Security validation report.
- Observability completeness report.
- Outstanding risk register snapshot.
