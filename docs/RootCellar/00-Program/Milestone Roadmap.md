# Milestone Roadmap

Parent: [[RootCellar Master Plan]]

## Phase Timeline
- Phase A Core Viability: March 2, 2026 to June 21, 2026.
- Phase B Replacement Credibility: June 22, 2026 to October 11, 2026.
- Phase C Enterprise Platform: October 12, 2026 to February 14, 2027.

## Current Milestone Status (February 28, 2026)
- M0 Foundation Ready: In progress.
  Linkage: [[Execution Plan Board]], [[Execution Status]]
  Completed inside M0 scope: engine/CLI/telemetry baseline.
  Remaining inside M0 scope: UI shell startup and CI artifact upload implementation.
- M1 and beyond: Not started.
  Linkage: [[Execution Plan Board]]

## Milestone M0 Foundation Ready (March 15, 2026)
- Engine skeleton compiles with workbook model baseline.
- UI shell renders virtualized grid prototype.
- Telemetry SDK integrated in UI, engine, scripting host stubs.
- CI executes lint + unit tests + artifact upload path.

## Milestone M1 Core Viability Alpha (May 10, 2026)
- Open/edit/save XLSX for baseline corpus with preserve mode.
- Formula parser + dependency graph + incremental recalc baseline.
- Editing loop (selection, input, copy/paste, undo/redo) stable.
- Python macro runner alpha with explicit permission prompts.
- Core observability dashboards live for error/latency/fidelity.

## Milestone M2 Core Viability Beta (June 21, 2026)
- No-repair pass on 80% of corpus.
- Headless CLI can run macro, recalc, export, emit JSONL report.
- Compatibility panel live with supported/partial/preserved matrix.
- Repro record/check workflow available in CLI.

## Milestone M3 Replacement Credibility Beta (August 30, 2026)
- Broad formula parity in scoped function families.
- Pivot view/refresh and chart edit for common types.
- VBA migration assistant produces classified reports + stubs.
- Signed add-in install flow and policy checks available.

## Milestone M4 Release Candidate (October 11, 2026)
- >= 92% corpus no-repair pass.
- Security hardening complete and validated.
- Performance SLOs consistently met in staging.
- Support playbook and incident workflows validated.

## Milestone M5 Enterprise Platform GA Candidate (February 14, 2027)
- Managed add-in repository support.
- Policy-driven trust modes and signed-only enforcement.
- Collaboration architecture RFC and serializable mutation protocol freeze.
- Long-term support branch and upgrade policy formalized.
